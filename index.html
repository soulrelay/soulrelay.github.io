客户端

使用RxJava和Retrofit搭配完成基本的登录和注册功能

  • RxJava：一个在 Java VM 上使用可观测的序列来组成异步的、基于事件的程序的库（说到根上，它就是一个实现异步操作的库）
  • Retrofit: 类型安全的网络库，封装了okhttp
